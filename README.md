# StartupOS

India's first AI-powered startup advisor.
5 AI agents that plan, debate, and learn
— autonomously.

## Tech Stack
- Frontend: Next.js 15, Tailwind, Shadcn/UI
- Backend: FastAPI, LangGraph, Groq API
- Database: Supabase (PostgreSQL + pgvector)
- AI: LLaMA 3.3 70B via Groq

## Features
- AI blueprint generation (5 agents)
- Multi-agent debate engine
- India-specific competitor analysis
- Investor pitch deck generation
- Vector-based memory system
- Ask your agents chat

## Deployment
- Frontend: Vercel
- Backend: Railway
- Database: Supabase
